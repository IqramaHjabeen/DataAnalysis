## Overview
This project analyzes the Canadian data analyst job market to understand trends, demand, and salary patterns.

## Objectives
- Analyze common job titles for data analysts in Canada
- Identify remote vs on-site job opportunities
- Determine most in-demand skills
- Analyze salary trends based on seniority levels
- Study job distribution across provinces
- Analyze industry-wise job opportunities
- Identify highest-paying data analyst roles

## Dataset
Job posting data related to data analyst positions in Canada.

- Pandas, NumPy
- Data Visualization (Matplotlib / Seaborn)
- Excel / Azure
